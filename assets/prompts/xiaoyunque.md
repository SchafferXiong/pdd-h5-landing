# 小云雀视频生成提示词包（与即梦同规格）

## 项目目标
视频并非独立播放器，而是H5页面联动素材。请保持强节奏、强利益点表达，避免冗余叙事。

## 参数
- 画幅：9:16
- 分辨率：1080x1920
- 时长：4秒主片 + 8秒补片
- 输出：MP4（H.264）

## 4秒主片 Prompt
拼多多品牌红白金电商促销风格。镜头快速推进礼盒，连续出现“百亿补贴”“低价秒杀”“天天领现金微信秒到账”视觉符号，最后0.8秒聚焦到高亮下载行动区域，形成强转化冲刺。画面明亮、干净、冲击感强，适配移动端首屏循环播放。

## 8秒补片 Prompt
拼多多品牌视觉延续。用8秒分镜说明四个利益点：万人团一件批发价、商家入驻0佣金千亿扶持、买菜返券、全域优惠收束。镜头平滑切换，信息可读性优先，方便在H5第二屏与利益点卡片同步。

## 负向约束
不要真人面部，不要品牌冲突，不要水印，不要暗黑色系，不要复杂长文本，不要卡顿抖动。
