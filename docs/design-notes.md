## 视频与页面联动策略
- `hero-main-4s.mp4` 绑定首屏状态机，按 4 个 beat 同步标题、副文案、CTA强度。
- `benefit-8s.mp4` 在第二屏进入视窗后播放，并同步高亮利益点卡片。
- `setHeroBeat(index)` / `setBenefitBeat(index)` / `setCtaIntensity(level)` 作为统一对外控制接口。
