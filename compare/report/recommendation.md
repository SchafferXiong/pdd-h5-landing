# 推荐结论

## 单技能推荐
`frontend-design-masterclass`

原因：
1. 在同素材前提下，镜头感、舞台层次与动效叙事最强，最符合“高交互 + 视频化”目标。
2. 首屏冲击力最足，能在 4 秒窗口内更快建立记忆点。
3. 对后续接入真实短视频（4s 主片 + 8s 补片）的结构兼容性最好。

## 组合推荐
`frontend-design-masterclass + ux-designer`

组合方式：
1. 以 `frontend-design-masterclass` 作为视觉与动效主框架。
2. 引入 `ux-designer` 的可用性策略：
- CTA层级更明确（主次按钮权重）
- 文案可读性阈值控制（对比度/字号）
- 弱网与低性能设备降级策略更激进
3. 输出“强视觉不牺牲转化路径”的投放版本。

## 投放建议
1. A 组：`frontend-design-masterclass`（极致吸引）
2. B 组：`ux-designer`（极致清晰）
3. 通过 4 秒点击率、首屏停留、CTA 点击到跳转率决定最终合并版本。
